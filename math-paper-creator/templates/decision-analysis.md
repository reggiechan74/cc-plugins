# {title}

## Outline

### Introduction
Decision context, the decision-maker, available alternatives, and
why a structured analysis framework is needed.
**Symbols:** none

### Alternatives
The set of options under consideration. Define the decision space
and any structural relationships between alternatives.
**Symbols:** Set

### Criteria and Weights
Evaluation dimensions and their relative importance. Attribute
definitions, weight elicitation method, and normalization.
**Symbols:** Set, Parameter

### Outcome Assessment
How each alternative performs on each criterion. Payoff matrices,
probability-weighted outcomes, or scoring functions.
**Symbols:** Parameter, Variable

### Ranking Method
The aggregation rule combining criteria scores into an overall
ranking — weighted sum, outranking, or expected utility.
**Symbols:** Expression, Objective

### Decision Rule
Conditions for selecting the preferred alternative. Dominance,
threshold requirements, or risk constraints.
**Symbols:** Constraint

### Sensitivity Analysis (optional)
How the ranking changes with perturbations to weights, scores,
or probabilities. Breakeven analysis and robustness checks.
**Symbols:** Expression, Parameter

### Conclusion
Recommended alternative, key trade-offs, and caveats about the
analysis assumptions.
**Symbols:** none
